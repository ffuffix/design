Code Block

<codeblock lang="…"> wraps a normal <pre><code>. A copy-to-clipboard button is part of the default anatomy — the script below injects one into every block; opt out per block with nocopy, as the diff sample does. Token colors target Prism, highlight.js, and Shiki output — this sample is hand-tokenized with Prism classes to show the palette without a bundled highlighter. 

async function sync(project) {
  const res = await fetch(`/api/projects/${project.id}`);
  if (!res.ok) throw new Error("sync failed");
  return res.json(); // hydrate the cache
}
